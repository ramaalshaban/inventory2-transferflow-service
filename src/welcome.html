Welcome to inventory-transferFlow Service

Handles internal inventory transfers (initiate/modify/track/cancel) between branches/sections. Tracks status, records audit log/history, and synchronizes transfer actions with per-location inventory.

Available Routes

  • getInternalTransfer => GET /internaltransfers/:internalTransferId
  • getInternalTransferByStatus => GET /internaltransferbystatus/:internalTransferId
  • createInternalTransfer => POST /internaltransfers
  • updateInternalTransfer => PATCH /internaltransfers/:internalTransferId
  • deleteInternalTransfer => DELETE /internaltransfers/:internalTransferId
  • listInternalTransfers => GET /internaltransfers
  • getInternalTransferLine => GET /internaltransferlines/:internalTransferLineId
  • createInternalTransferLine => POST /internaltransferlines
  • updateInternalTransferLine => PATCH /internaltransferlines/:internalTransferLineId
  • deleteInternalTransferLine => DELETE /internaltransferlines/:internalTransferLineId
  • listInternalTransferLines => GET /internaltransferlines
  • getInternalTransferLog => GET /internaltransferlogs/:internalTransferLogId
  • createInternalTransferLog => POST /internaltransferlogs
  • updateInternalTransferLog => PATCH /internaltransferlogs/:internalTransferLogId
  • deleteInternalTransferLog => DELETE /internaltransferlogs/:internalTransferLogId
  • listInternalTransferLogs => GET /internaltransferlogs
